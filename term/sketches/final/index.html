Phase 3b Term Project Sketches (final)

Data Description

The data I collected is about the effect of different study methods on memory. I tracked several key variables to explore the relationships between them.

  • Main Hypothesis: My initial hypothesis is that confidence level, study time, and study method may all impact the delayed recall score.

Sketch 1: Bubble Plot

Hypothesis: As my self-rated confidence increases, my delayed recall score increases.

View Sketch: sketch1.pdf

Data Representation:

  • X-Axis: confidence (0-10 rating)
  • Y-Axis: delayedRecallScore (0-15 rating)
  • Size: studyTime (18-33 mins)
  • Color: method (4 categories)

Sketch 2: Grouped Scatterplot

Question: Which study method has the greatest impact on the delayed recall score?

View Sketch: sketch2.pdf

Units and Approximate Range:

  • Method: Categorical (4 types)
  • Delayed Recall Score: 0-15 rating (Range: 7-12)
  • Study Time: Minutes (Range: 18-33)

Data Representation:

  • X-Axis: method (Categorical lanes)
  • Y-Axis: delayedRecallScore
  • Size: studyTime

Sketch 3: Parallel Coordinates Plot

Question: Is there a consistent trend or correlation across study time, confidence, and recall score for specific study methods? This chart helps analyze the multivariate relationship simultaneously.

View Sketch: sketch3.pdf

Units and Approximate Range:

  • Method: Categorical (4 types)
  • Delayed Recall Score: 0-15 rating (Range: 7-12)
  • Study Time: Minutes (Range: 18-33)
  • Confidence: 0-10 rating (Range: 7.0-8.5)

Data Representation:

  • Axes: Four vertical axes representing Method, Confidence, Study Time, and Recall Score.
  • Lines: Each line represents one study session connecting the four axes.
  • Color: Lines are colored by Method to distinguish groups.

Design Resources & Credits

  • The Data Visualisation Catalogue (https://datavizcatalogue.com): Used to research chart types for "Comparing" distributions. This verified that the Beeswarm Plot (Sketch 2) is suitable for comparing my categorical method data against recall scores.
  • From Data to Viz (https://www.data-to-viz.com): Used the decision tree for "Multivariate" data. This guided the decision to use a Parallel Coordinates Plot (Sketch 3) to visualize the relationships between all four variables (Method, Time, Confidence, Score) simultaneously, resolving the analyzability issues of my previous draft.
